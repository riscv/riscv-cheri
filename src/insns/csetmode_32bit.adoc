==== CSETMODE

ifdef::cheri_v9_annotations[]
NOTE: *CHERI v9 Note:* This instruction used to be CSetFlags.
endif::[]

include::new_encoding_note.adoc[]

Synopsis::
Capability set CHERI execution mode

Mnemonic::
`csetmode cd, cs1, rs2`

Encoding::
include::wavedrom/csetmode.adoc[]

Description::
Copy `cs1`  to `cd`  and set `cd.M` (the mode bit) to the least significant bit of
`rs2` . `cd.tag` is set to 0 if `cs1`  is sealed.

{cheri_mode_ext_name}
