==== {ADD_UW_CHERI} ({cheri_base64_ext_name})

Synopsis::
Add unsigned word for address generation

Mnemonic::
`{ADD_UW_CHERI_LC} {creg_pref}d, rs1, {creg_pref}s2`

Encoding::
[wavedrom, , svg,subs=attributes+]
....
{reg:[
    { bits:  7, name: 0x3b, attr: ['OP-32'] },
    { bits:  5, name: '{creg_pref}d' },
    { bits:  3, name: 0x1, attr: ['rv64: {ADD_UW_CHERI}'] },
    { bits:  5, name: 'rs1' },
    { bits:  5, name: '{creg_pref}s2' },
    { bits:  7, name: 0x04, attr: ['rv64: {ADD_UW_CHERI}'] },
]}
....

Description::
Copy the capability in `{creg_pref}s2` to `{creg_pref}d`.
+
Increment `{creg_pref}d.address` by the unsigned word in `rs1`.
+
Set `{creg_pref}d.tag=0` if `{creg_pref}s2` is sealed.
+
include::rep_range_check.adoc[]
+
include::malformed_cs2_clear_tag.adoc[]

+
