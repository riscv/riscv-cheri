[#SC,reftext="SC"]
==== SC

Synopsis::
Store capability

Mnemonic::
`sc ys2, offset(ys1)`

Encoding::
include::wavedrom/storecap.adoc[]

Store the YLEN-bit value in `ys2` to memory. The capability in `ys1`
authorizes the operation. The effective address of the memory access is
obtained by adding the address of `ys1` to the sign-extended 12-bit offset.
