include::wavedrom/cadd.adoc[]

NOTE: <<CMV>> uses the reserved encoding of <<CADD>> where `{cs2}=0`.

Description::
Copy the capability in register `{cs1}` to register `{cd}`.
+
For <<CADD>>,  increment `{cd}.address` by the value in `rs2[XLEN-1:0]` . +
For <<CADDI>>, increment `{cd}.address` by the sign-extended immediate value `imm`.
+
Set `{cd}.tag=0` if `{cs1}` is sealed.
+
