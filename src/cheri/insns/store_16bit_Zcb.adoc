Synopsis::
Stores (C.SH, C.SB), 16-bit encodings

{cheri_cap_mode_name} Mnemonics::
`c.sh rs2', offset({creg_pref}s1')` +
`c.sb rs2', offset({creg_pref}s1')`

{cheri_cap_mode_name} Expansions::
`sh rs2', offset({creg_pref}s1')` +
`sb rs2', offset({creg_pref}s1')`

{cheri_int_mode_name} Mnemonics::
`c.sh rs2', offset(rs1')` +
`c.sb rs2', offset(rs1')`

{cheri_int_mode_name} Expansions::
`sh rs2', offset(rs1')` +
`sb rs2', offset(rs1')`
