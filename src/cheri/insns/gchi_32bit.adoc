==== GCHI

include::new_encoding_note.adoc[]

ifdef::cheri_v9_annotations[]
NOTE: *CHERI v9 Note:* this instruction was called CGETHIGH.
endif::[]

Synopsis::
Capability get metadata

Mnemonic::
`gchi rd, cs1`

Encoding::
include::wavedrom/gchi.adoc[]

Description::
Copy the metadata (bits [CLEN-1:MXLEN]) of capability `cs1` into `rd`.
It is not required that the input capability `cs1`  has its tag set to 1.

+
