==== LC

Synopsis::
Load capability

Mnemonic::
`lc yd, offset(ys1)`

Encoding::
include::wavedrom/loadcap.adoc[]

Load a YLEN-bit value from memory and writes it to `yd`. The capability in `ys1` authorizes the operation. The effective address of the memory access is obtained by adding the address of `ys1` to the sign-extended 12-bit offset.
