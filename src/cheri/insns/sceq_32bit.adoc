==== SCEQ

include::new_encoding_note.adoc[]

ifdef::cheri_v9_annotations[]
NOTE: *CHERI v9 Note:* this instruction was called CSETEQUALEXACT.
endif::[]

Synopsis::
Set if Capabilities are EQual

Mnemonic::
`sceq rd, ys1, ys2`

Encoding::
include::wavedrom/sceq.adoc[]

Description::
`rd` is set to 1 if all bits (i.e. YLEN bits and the valid tag) of capabilities `ys1`
and `ys2`  are equal, otherwise `rd` is set to 0.

Operation::
+
