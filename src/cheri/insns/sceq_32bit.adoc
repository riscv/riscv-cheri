{SCEQ_DESC}

Mnemonic::
`{sceq_lc} rd, {creg_pref}s1, {creg_pref}s2`

Encoding::
include::wavedrom/sceq.adoc[]

Description::
Set `rd` to 1 if all bits (i.e., YLEN bits and the {ctag}) of capabilities `{creg_pref}s1`
and `{creg_pref}s2`  are equal, otherwise set `rd` to 0.
