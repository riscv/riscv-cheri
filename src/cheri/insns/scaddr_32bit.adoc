==== SCADDR

Synopsis::
Capability set address

Mnemonic::
`scaddr cd, cs1, rs2`

Encoding::
include::wavedrom/scaddr.adoc[]

Description::
Copy the capability `cs1` to `cd`.
+
Set `cd.address` to `rs2`.
+
Set `cd.tag=0` if `cs1` is sealed.
+
