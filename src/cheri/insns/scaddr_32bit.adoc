==== SCADDR

include::new_encoding_note.adoc[]

ifdef::cheri_v9_annotations[]
NOTE: *CHERI v9 Note:* this instruction was called CSETADDR.
endif::[]

Synopsis::
Capability set address

Mnemonic::
`scaddr yd, ys1, rs2`

Encoding::
include::wavedrom/scaddr.adoc[]

Description::
Set the address field of capability `ys1` to `rs2` and write the output capability to `yd`.
+
Set `yd.tag=0` if `ys1.tag=0` or `ys1`  is sealed.
+

+
