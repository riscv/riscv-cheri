[#AMOSWAP_CAP,reftext="AMOSWAP{LD_ST_DOT_CAP}"]
==== AMOSWAP{LD_ST_DOT_CAP}

Synopsis::
{AMOSWAP_CAP_DESC}

Mnemonic::
`AMOSWAP{LD_ST_DOT_CAP} {cd}, {cs2}, 0({cs1})`

Encoding::
include::wavedrom/amoswap_cap.adoc[]

include::load_store_creg0.adoc[]

Description::
Atomic swap of capabilities, authorized by the capability in `{cs1}`.
+
The operation is equivalent to an atomically executed sequence of:
+
`<<LOAD_CAP,{load_cap_name}>> {cd}, 0({cs1})`
+
`<<STORE_CAP,{store_cap_name}>> {cs2}, 0({cs1})`
+
`{cd}` is only updated if no exceptions are raised.
The value stored to memory is the original value of `{cs2}`, even if `{cd}={cs1}` or `{cd}={cs2}`.

include::atomic_exceptions.adoc[]

Prerequisites::
Zaamo, {cheri_base_ext_name}
