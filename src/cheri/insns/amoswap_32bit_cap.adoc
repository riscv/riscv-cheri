Description::
Atomic swap of capability type, authorized by the capability in `{creg_pref}s1`.
+
The operation is equivalent to an atomically executed sequence of:
+
`{load_cap_name_lc} {creg_pref}d, 0({creg_pref}s1)`
+
`{store_cap_name_lc} {creg_pref}s2, 0({creg_pref}s1)`
+
With the proviso that `{creg_pref}d` is only updated if no exceptions are raised.
