include::wavedrom/schi.adoc[]

Description::
The {SCHI_BASE} instruction packs the least-significant XLEN-bits of `rs1` and `rs2` into the YLEN least-significant bits of rd, zero extending the rest of `rd` and setting `rd.tag=0`.

==== {SCHI}

Synopsis::
Capability set metadata

Mnemonic::
`{schi_lc} cd, cs1, rs2`

Encoding::
{SCHI} is a pseudoinstruction for {SCHI_BASE}

Description::
Copy `cs1`  to `cd`.
+
Replace the capability metadata of `cs1` (i.e., bits [YLEN-1:XLEN]) with `rs2` and set `cd.tag` to 0.
