==== SCHI

include::new_encoding_note.adoc[]

ifdef::cheri_v9_annotations[]
NOTE: *CHERI v9 Note:* this instruction was called CSETHIGH.
endif::[]

Synopsis::
Capability set metadata

Mnemonic::
`schi yd, ys1, rs2`

Encoding::
include::wavedrom/schi.adoc[]

Description::
Copy `ys1`  to `yd` , replace the capability metadata (i.e. bits
[YLEN-1:MXLEN]) with `rs2`  and set `yd.tag` to 0.

Operation::
+
