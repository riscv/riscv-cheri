[#SCHI_BASE,reftext="{SCHI_BASE}"]
==== {SCHI_BASE}

Synopsis::
{SCHI_BASE_DESC}

Mnemonic::
`{schi_base_lc} rd, rs1, rs2`

Encoding::
include::wavedrom/schi.adoc[]

Description::
The {SCHI_BASE} instruction packs the least-significant XLEN-bits of `rs1` and `rs2` into `rd`, and sets `rd.tag=0`.

Capability set metadata

`{schi_lc} {creg_pref}d, {creg_pref}s1, rs2`

{SCHI} is a pseudoinstruction for {SCHI_BASE}

Copy `{creg_pref}s1`  to `{creg_pref}d`.
+
Replace the capability metadata of `{creg_pref}s1` (i.e., bits [YLEN-1:XLEN]) with `rs2` and set `{creg_pref}d.tag` to 0.
