== The {rv32y_uni_base_name} Capability Base for {cheri_base32_ext_name}

This section describes an in-memory format and properties of a capability encoding intended for {cheri_base32_ext_name}.
This format is based upon the <<rv64y_cap_description,{rv64y_uni_base_name}>> format and so only the changes relative to that format are listed here.

[#section_cap_encoding32]
=== Capability Encoding

The encoding format of the {rv32y_uni_base_name} capability is shown in xref:cap_encoding_xlen64[xrefstyle=short].

.Capability encoding for {rv32y_uni_base_name}
[#cap_encoding_xlen32]
include::img/cap-encoding-xlen32.edn[]

The parameters defined in <<rvy64_uni_base_name_param_summary>> are set as shown in <<rvy32_uni_base_name_param_summary>>.

.{rv32y_uni_base_name} parameter summary
[#rvy32_uni_base_name_param_summary,width="100%",cols="1,2,4",options=header,]
|==============================================================================
| Parameter | Value                | Comment
| MW        | {cap_rv32_mw_width}  | Mantissa width
| EW        | {cap_rv32_exp_width} | Exponent width
| CAP_MAX_E | {cap_rv32_exp_max}   | Maximum exponent value
| enableL8  | 1^1^                 | Whether the encoding format includes the L~8~ bit
|==============================================================================

^1^ enableL8 is in the description of the <<rv64y_cap_description,{rv64y_uni_base_name}>> encoding format even though that encoding format does not make use of it.
